# neovie.dev Landing Page v3

Moderne, professionelle Landing Page für neovie.dev (2026 Edition)

## Design-Philosophie

**Resonant Stark + Frosted Touch Hybrid**

- **Primary Aesthetic:** Resonant Stark (Emotional Minimalism)
  - Clean, intentionally simplified layouts
  - Ultra-thin fonts, soft gradients, lots of whitespace
  - Calm, elegant, purposeful
  
- **Secondary Aesthetic:** Frosted Touch (Glassmorphism 2.0)
  - Frosted glass panels, translucent layers
  - Diffused shadows, rounded edges
  - Adds depth without clutter

## Brand Guidelines

- **Primary Color:** Coral `#FF825C`
- **Secondary Color:** Dark `#1A1A2E`
- **Accent Color:** Teal `#4ECDC4`
- **Typography:** Poppins (headings) + Inter (body)

## Features

- ✅ Mobile-first responsive design
- ✅ WCAG AA compliant accessibility
- ✅ Performance optimized (< 2s FCP target)
- ✅ Smooth scroll-triggered animations
- ✅ Frosted glass effects with backdrop blur
- ✅ Interactive micro-animations
- ✅ Contact form with validation
- ✅ Accordion FAQ section
- ✅ Sticky navigation with blur effect

## Tech Stack

- HTML5 (Semantic)
- Tailwind CSS (via CDN for simplicity)
- Vanilla JavaScript (no dependencies)
- Google Fonts (Poppins, Inter)

## Deployment

Deployed via GitHub Pages

## Local Development

Simply open `index.html` in a browser, or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve .
```
